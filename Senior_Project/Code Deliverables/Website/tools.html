Getting Started

NOTE: YOU NEED TO INSTALL THE FOLLOWING PROGRAMS IN THE ORDER THAT THEY ARE PRESENTED.
FAILURE TO DO THIS MIGHT RESULT IN PULSERS NOT WORKING AS EXPECTED.

Installing NodeJS

  1. 1. First, you will need to download the latest version of the NodeJS Framework which corresponds to your operating system.
    The different versions for this installer can be found here.
  2. 2. Once downloaded, locate the installer and open it.
  3. 3. Click on "Next", then be sure to read the terms and conditions and accept these.
  4. 4. Click on "Next", and then click on "Next" again to confirm the path of installation.
  5. 5. Next, you will be prompted with a window showing a few options to choose from. Make sure to select the "npm package manager"
    then click on "Next".

  6. 6. You will then be prompted if you would like to install additional tools with this package. Make sure the checkbox is NOT selected
    and click "Install".
  7. 7. Wait for the program to finish installing and once it is done click on "Finish".

Installing Serial Port Server

  1. 1. Now, select the "Serial Port Server" installer version which corresponds to your operating system from the choices below.
    (Currently Windows only available)

  2. 2. After the file has downloaded, locate and open it.
  3. 3. Once it opens, it will ask if you trust the source. For Windows users, select "More Info" and then "Run Anyways" on the bottom.
  4. 4. After this, press on "Yes" and then click on "Install".
  5. 5. Wait for the program to finish installing and after it is done click on "Close".
  6. 6. You should now see an executable called "Serial Port Server" on your Desktop. Go ahead and open this application.
  7. 7. You will be prompted if you would like to begin a connection. Before you click "Connect", be sure to first connect your
    pulser's box to the computer. Once the box is connected, click on "Connect".
  8. 8. That's it! The next step is to create an account. Make sure to always open the "Serial Port Server" application before
    starting a session.

  9. The next step is to create an account if you haven't, otherwise you may log in to your account to start your first session.
    Click here to be redirected to the sign-up page to open a new account.
