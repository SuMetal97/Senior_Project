Frequently Asked Questions (FAQs)

  1. Question: Why are my pulsers not vibrating when I am in a session with a therapist?
  2. Answer: There are many possible reasons why the pulsers are not working. The following are a list of possible solutions to this problem:
    1. Make sure the "control box" connected to the computer.
    2. You must open the "Serial Port Server" application and press on "Connect" after connecting the "control box" to the computer.
    If you have the server running, please press "Disconnect", then ensure the "control box" of the pulsers is connected before clicking
    "Connect" once more.
    3. When installing the tools required to run the pulsers, you must make sure that the "Serial Port Server" is installed AFTER installing "NodeJS".
